# NamespacedMixin

Namespaces your mixins. Proper Composition.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'namespaced_mixin'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install namespaced_mixin
